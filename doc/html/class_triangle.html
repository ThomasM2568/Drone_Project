Drone Project
This project models a network of drones and servers within a geometric environment where communication and movement are based on a graph structure. The primary objective is to simulate drone routing and interaction with servers, subject to spatial constraints and communication rules. Drones are tasked with navigating a network of servers, and only the first drone to arrive at each server will continue its route. The others will wait their turn, landing at the server until it is their turn to proceed.

Detailed Description

The Triangle class represents a triangle in a 2D plane defined by three vertices. It provides methods for geometric operations such as checking if a point is inside the triangle, computing the circumcircle, and checking the Delaunay property.

◆ checkIsOpen()

void Triangle::checkIsOpen ( QVector< Triangle * > triangles)
inline

Checks if the triangle is open based on its shared edges with other triangles.

Parameters
trianglesA list of triangles to check against for shared edges. This function checks the edges of the current triangle and determines whether each edge is shared with another triangle. If at least one edge is not shared, the triangle is considered "open" and is added to the openSides set. It updates the isOpen status and inserts non-shared edges into the openSides set.
