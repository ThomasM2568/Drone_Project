Drone Project
This project models a network of drones and servers within a geometric environment where communication and movement are based on a graph structure. The primary objective is to simulate drone routing and interaction with servers, subject to spatial constraints and communication rules. Drones are tasked with navigating a network of servers, and only the first drone to arrive at each server will continue its route. The others will wait their turn, landing at the server until it is their turn to proceed.

Detailed Description

The Canvas class allows to draw the graphical interface canvas content.

Author
Kyllian Cuevas & Thomas Mirbey
Date
19/11/2024
Todo
Canvas related methods

It stores:

  • a list of triangles : QVector<Triangle*>;
  • a list of vertices : QVector<Vector2D> vertices;
  • a list of polygons : QVector<MyPolygon*>;
  • the scale = 0.5 (float);
  • the origin = QPoint(10, 10) (QPoint);

It is used by the MainWindow class to create the data visualization.
